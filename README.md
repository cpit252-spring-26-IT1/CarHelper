# CarHelper

## Description 
CarHelper is a web app we are building to help car owners easily figure out what is wrong with their cars, understand those confusing dashboard warning lights, and get a clear idea of repair and resale prices.


## Features
- ***Image-Based Diagnosis:*** Users take a photo of a warning light, and the AI instantly identifies the problem.
- ***Repair Cost Estimation:**** Estimates repair prices for the Saudi market so users do not pay too much.
- ***Car Resale Valuation:*** Calculates a fair resale price based on the car's current shape.
