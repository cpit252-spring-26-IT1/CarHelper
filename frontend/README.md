# CarHelper Frontend

This is the frontend part of the CarHelper project.

## What it does

The frontend allows users to:

- Create an account
- Login
- Upload a car image for diagnosis
- Describe car problems
- Estimate repair cost
- Estimate car resale value
- View previous results in the profile page
- Switch between Arabic and English
- Switch between light mode and dark mode

## Technologies

- React.js
- CSS
- i18next
